## HTML Tutorial
A simple tutorial explaining all important HTML topics with examples.

## HTML HOME
HTML stands for **HyperText Markup Language**. It is used to create web pages and web applications.

## HTML Introduction
HTML describes the **structure of a webpage** using markup. Elements are represented by **tags**.

## HTML Editors
Popular editors to write HTML:
- Visual Studio Code
- Sublime Text
- Atom
- Notepad++

## HTML Basic
Every HTML document starts with:
```html
<!DOCTYPE html>
<html>
<head></head>
<body></body>
</html>
```

## HTML Elements
HTML elements are the building blocks of HTML pages. Example:
```html
<p>This is a paragraph.</p>
```

## HTML Attributes
Attributes provide **additional information** about elements. Example:
```html
<a href="https://example.com">Visit</a>
```

## HTML Headings
HTML provides six levels of headings:
```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
```

## HTML Paragraphs
Example of a paragraph:
```html
<p>This is a paragraph.</p>
```

## HTML Styles
Example of styling text:
```html
<p style="color:red;">This is a styled paragraph (red color).</p>
```

## HTML Formatting
Formatting tags:
- `<b>Bold</b>`
- `<i>Italic</i>`
- `<u>Underline</u>`
- `<mark>Highlighted</mark>`

## HTML Quotations
Block level and inline quotes:
```html
<blockquote>This is a blockquote.</blockquote>
<q>This is an inline quotation.</q>
```

## HTML Comments
HTML comment example:
```html
<!-- This is a comment -->
```

## HTML Colors
Setting text color:
```html
<p style="color:blue;">Blue text</p>
```

## HTML CSS
Applying inline CSS:
```html
<p style="font-family:Arial; font-size:20px;">Styled with inline CSS.</p>
```

## HTML Links
Creating hyperlinks:
```html
<a href="https://www.example.com" target="_blank">Visit Example</a>
```

## HTML Images
Displaying an image:
```html
<img src="https://via.placeholder.com/150" alt="Placeholder Image">
```

## HTML Favicon
Adding a favicon in the `<head>` section:
```html
<link rel="icon" href="favicon.ico" type="image/x-icon">
```

## HTML Page Title
Defining the page title inside the `<title>` tag.

## HTML Tables
Creating a simple table:
```html
<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>Ali</td><td>24</td></tr>
</table>
```

## HTML Lists
Unordered and ordered lists:
```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>
```

## HTML Block & Inline
Examples:
```html
<div>This is a block element (div).</div>
<span>This is an inline element (span).</span>
```

## HTML Div
Example with styling:
```html
<div style="background-color: lightgrey;">This is a div container.</div>
```

## HTML Classes
Using classes:
```html
<div class="myclass">This is a div with a class.</div>
```

## HTML Id
Using an id:
```html
<div id="myid">This is a div with an ID.</div>
```

## HTML Iframes
Embedding another page:
```html
<iframe src="https://example.com" width="300" height="200"></iframe>
```

## HTML JavaScript
Adding JavaScript inside HTML:
```html
<button onclick="alert('Hello!')">Click Me</button>
```

## HTML File Paths
Examples of relative and absolute paths:
```html
src="images/pic.jpg" (relative path)
src="https://www.example.com/images/pic.jpg" (absolute path)
```

## HTML Head
The head section contains meta info, links to CSS/JS, title, etc.

## HTML Layout
HTML layout tags:
```html
<header>Header Area</header>
<nav>Navigation Links</nav>
<section>Content Section</section>
<aside>Sidebar</aside>
<footer>Footer Info</footer>
```

## HTML Responsive
Making a page responsive:
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
And using CSS media queries.

## HTML Computercode
Displaying computer code:
```html
<code>console.log("Hello World");</code>
```

## HTML Semantics
Semantic tags:
```html
<article>This is an article element.</article>
<section>This is a section element.</section>
```

## HTML Style Guide
Writing clean and consistent HTML:
- Use lowercase tags
- Properly indent your code

## HTML Entities
Displaying special characters:
```html
<p>Use &lt; and &gt; to display &lt;HTML&gt; symbols.</p>
```

## HTML Symbols
Common symbols:
```html
<p>&copy; &euro; &yen;</p>
```

## HTML Emojis
Examples of emojis:
```html
<p>&#128512; &#128525; &#128640;</p>
```

## HTML Charsets
Using UTF-8 charset:
```html
<meta charset="UTF-8">
```

## HTML URL Encode
In URL encoding:
- A space character is encoded as `%20`.

## HTML vs. XHTML
- XHTML is stricter than HTML.
- XHTML requires elements to be properly nested and closed.

# Forms Section

## HTML Forms
Creating a basic form:
```html
<form action="#">
  <input type="text" placeholder="Name"><br><br>
  <input type="submit" value="Submit">
</form>
```

## HTML Form Attributes
Form attributes:
- `action`
- `method`
- `target`
- `autocomplete`

## HTML Form Elements
Form with input, select, and textarea:
```html
<form action="#">
  <label for="username">Username:</label>
  <input type="text" id="username" name="username"><br><br>

  <label for="country">Country:</label>
  <select id="country" name="country">
    <option value="pakistan">Pakistan</option>
    <option value="india">India</option>
  </select><br><br>

  <textarea rows="4" cols="30" placeholder="Enter text here..."></textarea>
</form>
```

## HTML Input Types
Different input types:
```html
<input type="text" placeholder="Text"><br>
<input type="password" placeholder="Password"><br>
<input type="email" placeholder="Email"><br>
<input type="date"><br>
<input type="file"><br>
<input type="color"><br>
<input type="range"><br>
<input type="number"><br>
```

## HTML Input Attributes
Input attributes:
```html
<input type="text" value="Readonly" readonly><br>
<input type="text" value="Disabled" disabled><br>
<input type="text" placeholder="Required Field" required>
```

## Input Form Attributes
Form with autofocus and autocomplete:
```html
<form action="#" autocomplete="on">
  <input type="text" name="name" autofocus><br><br>
  <input type="submit">
</form>
```
